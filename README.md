# Alpha Neurofeedback: Hybrid vs. Intermittent Feedback

(Work in progress)

This repository contains the code and materials necessary to run the neurofeedback experiment described in [TBD], and to reproduce the analyses and figures reported in the manuscript.

# Contents

* code/experiment — scripts to run the experiment (stimulus presentation, online EEG analysis, sonification).
* code/analysis — scripts to reproduce the statistical analyses and figures reported in the manuscript.
* documents — the experiment protocol as well as documents for the participants (general information, consent form, questionnaires)
* stimuli — audio files required to run the experiment.
* libs — third-party dependencies not redistributed here; see libs/README.md for requirement list.
